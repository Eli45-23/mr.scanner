# Options Whale Scanner Semantic Layer

Use this skill for source-backed scanner reviews.

## Start Here

1. Read `references/semantic-layer.md`.
2. Read the current scanner export or logs before making claims about a session.
3. Use event-level, not raw alert-row, counts for signal-quality conclusions.
4. Label incomplete outcome windows and unconfirmed next-day OI explicitly.
